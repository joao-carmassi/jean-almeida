## Análise Estratégica

### Persona do Paciente-Alvo

- **Quem:** Adultos entre 25–55 anos, moradores ou trabalhadores em São Paulo (especialmente região da Av. Paulista / Bela Vista / Paraíso)
- **Dor:** Sofrimento emocional sem diagnóstico claro, burnout corporativo, ansiedade, depressão, dificuldade de concentração (TDAH), insônia crônica
- **Barreira:** Medo de julgamento, experiências anteriores ruins com psiquiatria, dúvida sobre se realmente precisa de ajuda
- **Motivação:** Recuperar qualidade de vida, função no trabalho e nos relacionamentos

### Palavras-chave Primárias (alta intenção comercial)

- `psiquiatra em São Paulo`
- `psiquiatra particular São Paulo`
- `psiquiatra Av Paulista`
- `agendar consulta psiquiatra SP`
- `psiquiatra humanizado São Paulo`

### Palavras-chave Secundárias (condições + localização)

- `tratamento depressão São Paulo`
- `tratamento ansiedade São Paulo`
- `TDAH adulto psiquiatra SP`
- `transtorno bipolar tratamento São Paulo`
- `síndrome de burnout psiquiatra SP`
- `insônia crônica tratamento São Paulo`
- `esquizofrenia tratamento São Paulo`
- `psiquiatra Bela Vista São Paulo`
- `psiquiatra metrô Consolação SP`
- `saúde mental São Paulo`
- `consulta psiquiátrica online São Paulo`

### Diferencial Competitivo a Destacar

1. **Localização premium** — Av. Paulista, símbolo do coração de São Paulo, acessível por metrô
2. **Abordagem humanizada** — escuta ativa, sem julgamento, foco no ser humano antes do diagnóstico
3. **CRM registrado** — credibilidade e segurança técnica
4. **Multilíngue** — atendimento em Português e Espanhol (diferencial para comunidade latinoamericana em SP)

### Q&A

**Q: Qual a diferença entre psiquiatra e psicólogo?**

```
A: O psiquiatra é médico especializado em saúde mental — pode fazer diagnósticos, prescrever medicamentos e indicar exames. O psicólogo realiza psicoterapia, mas não prescreve medicação. Em muitos casos, o ideal é trabalhar com os dois profissionais em conjunto.
```

**Q: Preciso de encaminhamento médico para consultar?**

```
A: Não. Você pode agendar sua consulta diretamente, sem necessidade de encaminhamento. Basta entrar em contato pelo WhatsApp ou formulário.
```

**Q: O Dr. Jean atende pelo convênio?**

```
A: As consultas são realizadas de forma particular. Aceitamos cartões de crédito e débito. Muitos planos de saúde oferecem reembolso — consulte o seu.
```

**Q: É possível fazer consulta online?**

```
A: Sim, oferecemos consultas por telemedicina para pacientes que não podem comparecer presencialmente. Entre em contato para verificar disponibilidade.
```

**Q: Como sei se preciso de um psiquiatra?**

```
A: Se você está sofrendo — seja com tristeza, ansiedade, insônia, mudanças de humor ou dificuldades emocionais — já é motivo suficiente para buscar ajuda. Não espere a situação piorar.
```

**Q: O consultório fica em que bairro de São Paulo?**

```
A: O consultório está localizado na Av. Paulista, 2494 — conjunto 94, no bairro Bela Vista, São Paulo. Fácil acesso pelo metrô (estação Consolação ou Trianon-MASP).
```

**Q: O atendimento é feito em outro idioma além do português?**

```
A: Sim. O Dr. Jean Almeida atende também em Espanhol, o que é um diferencial importante para a comunidade latinoamericana residente em São Paulo.
```
